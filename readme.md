#  Bing主题模版
### 半年前还是小白的时候 写的💩山代码
### 注意：模版不适合纯小白，请谨慎切换（注意备份数据！！）  
---
## 使用方式
### 覆盖文件,config、public、resources 三个文件
### 后台切换主题
### 功能配置文件在config/bing.php里配置
---
## 功能
- banner图
- 新品上市、火爆商品、销售数量 显示 (自定义 火爆产品(数量) 新品上市(天数))
- 站长联系方式 目前支持tg和qq的图标
- 友情链接
